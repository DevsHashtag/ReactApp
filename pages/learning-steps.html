ما در reactapp.ir سعی می کنیم محتوای آموزشی را در اختیار شما کاربران عزیز قرار دهیم تا بتوانید مفاهیم javascript در front-end و back-end را به بهترین شکل ممکن فرا بگیرید و ضمن تبدیل شدن به یک Full Stack Developer وارد بازار کار شوید، به همین جهت گام های یادگیری را برای شما مشخص نموده ایم تا بتوانید به راحتی وارد دنیای javascript شوید.
راهنمای یادگیری

آموزش React

React یه کتابخونه جاوا اسکریپتیه که با استفاده از اون میتونین وب سایت و وب اپلیکیشن های فوق العاده سریع ایجاد کنین.

آموزش Nodejs

Nodejs یه پلتفرم بر مبنای موتور جاوا اسکریپته که برای پیاده سازی برنامه های Real-time مبتی بر داده استفاده می شود.

آموزش React Native

React Natice یه فریمورک جاوا اسکریپتیه که میتونین با اون و تنها یکبار کد زدن و خروجی native برای دو سیستم عامل ios و android بگیرین.

آموزش Javascript

جاوا اسکریپت یه زبان برنامه نویسی قدرتمنده که با استفاده از اون می تونین اپلیکیشن های موبایل، وب سایت، وب اپلیکیشن و برنامه های تحت ویندوز بنویسین.
