    • ایده این سایت از کجا اومد

    اوایل سال 96 بود که ما کار روی یک استارتاپ در زمینه گردشگری رو شروع کردیم. این استارتاپ قراره یک اپلیکیشن باشه که به مردم اماکن گردشگری ، سوغات و صنایع دستی رو بر اساس یک روش جدید معرفی کنه.چون ما استارتاپی هستیم و زمان خیلی برامون مهمه تصمیم گرفتیم اپلیکیشن رو با React Native توسعه بدیم تا هر دو نسخه android و ios رو با هم منتشر کنیم.اما برای یادگیری react native منبع فارسی اصلا وجود نداشت و ما مجبور بودیم که برای رفع اشتباهات و پاسخ گرفتن سوالات کلی جستجو کنیم و منابع انگلیسی بخونیم. اینجا بود که تصمیم گرفتیم reactapp.ir رو راه اندازی کنیم تا تجربیاتمون و راه های رفع مشکلاتی که بهش برخوردیم و برمیخوریم با شما به اشتراک بزاریم.

    ما چه میکنیم

    هدف اصلی ما آموزش react native و react js با زبان فارسی بود که حالا بعد از گذشت دو سال این وب سایت با کمک و حضور شما قوی تر از گذشته تصمیم گرفته به مرور خود را به عنوان یک مرجع آموزش فارسی جاوااسکریپت معرفی کند . ما برای تحقق این امر تصمیم گرفتیم که از چهار قسمت و بدنه اصلی استفاده کنیم:
    1. دوره های آموزشی : دوره هایی پروژه محور برای آموزش ساخت یک اپلیکیشن واقعی با react native وساخت وب اپلیکیشن با react js یا هر کدام از فریمورک های محبوب جاوااسکریپت
    2. آموزش متنی : مقالات آموزشی برنامه نویسی react native
    3. تله فیلم آموزشی : فیلم هایی کوتاه برای رفع اشتباهات متدوال در برنامه نویسی جاوااسکریپت و فریمورک های آن
    4. وبلاگ : مطالب آموزشی در مورد برنامه نویسی ، استارتاپ ، مدیریت و دنیای جذاب جاوااسکریپت
    خیلی خوشحالیم که به عنوان اولین مرجع آموزش فارسیreact native و react js درسال 96 شروع به فعالیت کردیم و امیدواریم که بتونیم به جامعه توسعه دهندگان فارسی زبان جاوااسکریپت کمک کنیم.
